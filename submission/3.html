Initial questions Which families are the biggest in each Class Mammalia (Mammals), Aves (Birds),Amphibians (Amphibia) and Reptiles (Reptilia)
Description:  This Bar chart shows which family, inside each of the chosen classes in Animalia, has the largest count. A Family is a grouping between Order and Genus in Taxonomic Rankings. There are no hard rules to belong inside a Family in the Zoological world just that each animal in a family is very closely related but not the direct same species.

Families can be used for evolutionary, palaeontological and genetic studies because they are more stable than lower taxonomic levels such as genera and species. For this reason being able to identify the ranking of families inside a class may be very useful.

As for me I thought the question was interesting as it is not an immediately attainable fact for someone who is not familiar with zoology sites.

Insight: Well Thanks to vega-lite's ability to include a tooltip we can readily find out which families are the biggest in each Class. The top Three Families in each class are as follows:

  • Mammalia

    1. Bovidae. Bovine Animals (Aka cloven-hoofed Animals)

    2. Vespertilionidae. microbats

    3. Cercopithecidae. Old World Monkeys

  • Aves

    1. Psittacidae. True Parrots

    2. Anatidae. Water Birds

    3. Accipitridae. strongly hooked billed birds

  • Amphibia

    1. Ranidae. True Frogs

    2. Salamandridae. True Salamanders

    3. Bufonidae. True Toads

  • Reptilia

    1. Colubridae. Colubrid Snakes

    2. Viperidae. Vipers

Also thanks to the nature of the bar chart we can see the Top 3 Biggest families over all.

  1. Psittacidae. True Parrots

Design considerations I have used a barchart as it is very effective to show this type of nominal data attached to a quantitative value. This ordered layout clearly shows me the data in a way that is easy to read with the help of a tooltip. The reason a Tooltip is used is because when dealing with a facet in vega-lite it will not show the values on each sub graph it will only show all the families on the bottom which is incredibly difficult to read. so I removed the label values on the bottom. This barchart shows us two useful pieces of information. The biggest families in each class and the biggest families across the classes.

I also considered using a bubble chart to represent the data where the size of the bubble is proportional to the bubble size and the color of the bubble would be the different classes. I decided not to use this as due to the amount of data we have we would have many many bubbles and it would be quite unwieldy.

Data filtering and transformation:  The data has all classes of animals and this question was specifically pertaining to some classes I had to scrape out just Mammalia, Aves, Amphibia and Reptilia just for this question.
